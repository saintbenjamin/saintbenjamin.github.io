Handles lazy loading and caching of Bible text and metadata from JSON sources.

class core.utils.bible_data_loader.BibleDataLoader(json_dir=None, text_dir=None)[source]

Bases: object

Loads and lazily caches Bible-related JSON data: version aliases, book aliases, canonical names, and full Bible texts.

Lazy-loading ensures performance is optimized by only loading data on-demand.
